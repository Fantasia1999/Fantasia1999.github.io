验证方式

ping可以验证吗

ping的协议不是https，也不是https，是ICMP协议，所以，不能单纯用ping通来判断可以进行http访问。

curl验证

curl -vv http://www.google.com，用这条命令来验证，如果返回如下结果表示代理设置成功。

powershell下好像无法使用这个命令，可以直接尝试curl www.google.com
